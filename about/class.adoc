=== 클래스 (class)
자바에서 클래스란 객체를 정의하는 틀 또는 설계도와 같은 의미로 사용됨.

객체의 상태를 나타내는 *필드(field)와* +
객체의 행동을 나타내는 *메소드(method)* 로 구성됨

[source,java]
----
class className {
    // 필드의 선언
    // 필드의 선언
    ...
    // 메소드의 선언
    // 메소드의 선언
    ...
}
----
